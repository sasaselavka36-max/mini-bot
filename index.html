Мини-Шахматы

Правила шахмат

Доска и фигуры

Игра ведётся на доске 8×8 клеток (64 клетки). Клетки чередуются по цвету. У каждого игрока есть 16 фигур:

  • 1 король
  • 1 ферзь
  • 2 ладьи
  • 2 слона
  • 2 коня
  • 8 пешек

Расстановка

  • Король — e1 для белых, e8 для чёрных
  • Ферзь — d1 для белых, d8 для чёрных
  • Ладьи — a1 и h1 для белых, a8 и h8 для чёрных
  • Кони — b1 и g1 для белых, b8 и g8 для чёрных
  • Слоны — c1 и f1 для белых, c8 и f8 для чёрных

Ходы

Игроки ходят по очереди. Первый ход делает белый. Ход — это передвижение фигуры на свободную клетку или взятие фигуры соперника. Взятая фигура снимается с доски. Ходы нельзя отменять, кроме случаев нарушения правил.

Цель игры

Поставить мат королю противника — создать ситуацию, когда король не может защититься и будет взят следующим ходом.

Окончание

  • Побеждает игрок, поставивший мат.
  • Партия может закончиться вничью: троекратное повторение позиции, недостаток материала для мата, соглашение игроков.
